# BioQCore.org v1.0.0-rc

Trust-first public website for BioQCore.

## Local preview

```bash
python3 -m http.server 8000
open http://localhost:8000
```

## Deploy to GitHub Pages

This repository is static-first. Upload files to `main` or use GitHub Desktop / GitHub web editor. `CNAME` is set to `bioqcore.org`.
