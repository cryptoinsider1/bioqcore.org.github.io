Mission

Human-Centric

Фокус на детях, пациентах и исследователях. BioQCore стремится сделать сложные технологические системы прозрачными, этичными и безопасными для реальных людей.

BioQCore Trust Fabric

Trust Fabric — это слой, где соединяются криптография, инфраструктура и человеческая ответственность. Он связывает Root CA, Trust Center, Edge-узлы и клиенты (iOS, веб, исследовательские платформы).

Root & Vault Layer

  • Root CA на Raspberry Pi + зашифрованный SSD
  • Оффлайн-режим · минимальное включение
  • Хранение мастер-ключей и политик

Trust Center Layer

  • Intermediate CA & issuing CA
  • GraphQL / REST Trust API
  • DID Registry · Audit · CRL / OCSP

Edge & Client Layer

  • MikroTik BR5009 как Edge-узел
  • VPN, TLS-терминация, mTLS
  • iOS BioQCore App · Web Clients

Roadmap · Phase 0 → Phase 3

Phase 0 — Foundations

Статический сайт, архитектурные документы, лабораторная PKI (Root CA на Raspberry, Intermediate CA, первые эксперименты с MikroTik и VPN).

Phase 1 — Trust Center Prototype

BioQCore Trust Center: GraphQL API, issuing CA, интеграция с iOS агентом и внутренними сервисами.

Phase 2 — Ecosystem Integration

Связка с WWBTC, CosmicFutureToDay, подключение лабораторий и партнёрских проектов.

Phase 3 — Production-grade Consortium

Масштабируемая экосистема с чёткой PKI, аудитацией, отказоустойчивостью и многоуровневой защитой.
